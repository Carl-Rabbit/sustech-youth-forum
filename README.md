# 基于Python的科研数据分析入门
南科新青年讲堂

## 分享内容
在科研工作中，非计算机专业的科研人员通常有对科研数据进行处理，分析以及可视化的需求，在本次新青年讲堂中，我们向大家讲解如何使用语法简单，功能强大的Python来解决上述问题，具体内容包括：Python介绍，Python编程环境的配置，在Python中进行数据清洗和处理的基本方法，如何搭建和训练机器学习模型，以及如何通过Python对数据进行可视化展示与分析。
